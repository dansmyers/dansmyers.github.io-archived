Conditional Statements

A conditional statement (or if statement) allows a block of code to execute only if a certain condition is satisfied.

Here's an example:


var a = 2;
var b = 1;
					
if (a > b) {
    a = b;
}

The if statement tests if certain condition is true. This example tests if a is greater than b, which is true.

If the condition is true, the statements in between the curly braces, { and }, are executed.

If the condition is false, the statements are not executed and the program simply continues to the next line after the if block.

Alternative Execution

The if-else statement has two possible paths. One of the two is chosen based on a test.

Here's an example:


var a = 1;
var b = 3;
				
if (a > b) {
	b = a;
} else {
	a = b;
}

If the condition evaluates to true, the code within the if block is executed and else block is skipped.

If the condition is false, the if block is skipped and the else block is executed.

In this case, the condition is false, so the else block executes and a will be set to the value of b.

Functions

A function is a block of code that can be called from another part of the program. Functions are frequently used for code that's going to be re-used multiple times, so it doesn't have to be copied over and over again

Here's an example:


function fun() {
	var p = 1;
	var q = 2;
	
	return p + q;
}
				
var a = fun();

The program defines a function named fun that contains four statements. The function's code does not execute immediately when the program runs, but only when it is called by antoher part of the program.

The line 


var a = fun();
calls the function, which causes the program to jump up to fun and begin executing its code.

The last line of fun returns a value back out to the rest of the program. In this case, the returned value is 3.

The returned value is then assigned to a, resulting in a being set to 3.

Function Arguments

A function can also take arguments that control its execution.

Here's an example:


function fun(p) {
	var q = p = 2;
	
	return p + q;
}
				
var a = 1;
var b = fun(a);

Here, function fun takes a single argument when it's called. In this example, the value of a, 1, is passed in to fun and assigned to the variable p.

p can be used like any other variable within the function, but it's initial value is controlled by the argument given when fun is called.

Loops

A loop is a statement that executes a block of code multiple times.

One basic type of loop is the for loop.

Here's an example:


var a = 1;

for (var i = 0; i < 2; i++) {
    a = a + 1;
}

The variable i is called the loop index variable. It's used like a counter to keep track of how many times the loop has executed.

The loop declaration has three parts.

First, an initialization, which assigns an initial value to i at the start of the loop. In this case, i is initialized to 0.

Second, a stopping condition. The loop continues to run until this condition becomes true. In this case, the loop will repeat until i becomes 2 or greater.

The third part specifies how the loop index variable i changes at the end of each pass through the loop. Here, i++ is a special shorthand to indicate that i increases by 1 on each trip through the loop body.

On the first pass through the loop, i is 0. The body of the loop executes, and adds 1 to a. At the end of the loop body, i is increased to 1, and the loop checks to see if the stopping condition is satisfied. It's not, so the loop repeats.

On the second pass, i is 1. The loop body executes and a is increased to 3. At the end of the loop body, i is increaesd again, becoming 2. The stopping condition is now satisfied, so the loop ends.

Loops

An second type of loop is the while loop.

Here's an example:


var a = 1;
var b = 3;

while (a < b) {
	a = a + 1;
}

The while loop continues to execute as long as its condition is true.

In this case, the condition is true when the loop begins, because 1 is less than 3, so the loop body executes.

The first pass through the loop increase a to 2. The condition is checked again, and it's still true (because 2 is less than 3), so the body executes another time.

On the second pass through the loop, a is increased to 3. The condition is now false, so the loop stops, and the body does not execute again.
